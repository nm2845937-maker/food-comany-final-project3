# 🍔 Food Company Website

## 📌 Project Description
This project is a simple and interactive food company website.  
It is designed to showcase food products and provide an easy and attractive user experience.

The website includes multiple sections such as Home, About, and Contact, along with a responsive navigation bar and a Dark Mode feature.

---

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript

---

## ✨ Features
- Responsive design (works on different screen sizes)
- Dark Mode toggle 🌙
- Navigation bar for easy browsing
- Clean and simple UI design

---

## 📂 Project Structure
- `index.html` → Main structure of the website  
- `style.css` → Styling and layout  
- `script.js` → Interactivity and functionality  

---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser
